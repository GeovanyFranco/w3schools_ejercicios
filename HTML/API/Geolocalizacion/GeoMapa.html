Insertar mapas o indicaciones

  1. Abre Google Maps en el ordenador.
  2. Accede a las indicaciones, al mapa o a la imagen de Street View que quieras compartir.
  3. En la parte superior izquierda, haz clic en Menú .
  4. Selecciona Compartir o insertar el mapa. Si no aparece esta opción, haz clic en Obtener enlace a este mapa.
    • Opcional: Marca la casilla situada junto a "URL corta" para crear un enlace de página web más corto.
  5. Copia y pega el enlace donde quieras compartir el mapa.

Comience con la plataforma Google Maps

Visión general

Google Maps Platform es un conjunto de API y SDK que se administran desde la consola Google Cloud (también conocida como Cloud Console). Para comenzar con Google Maps Platform, necesita:

  1. Crea una cuenta de facturación
  2. Crear un proyecto
  3. Habilite una o más API o SDK
  4. Obtener, agregar y restringir una clave API

Sus opciones para completar estos procedimientos dependen de si es un usuario nuevo o un usuario actual de Cloud Console.

Nuevo usuario : un nuevo usuario es un cliente que nunca ha usado la consola de nube para crear una cuenta de facturación o un proyecto. Si es un usuario nuevo, use el Inicio rápido o consulte los Procedimientos iniciales .

Usuario actual : Un usuario actual es un cliente que ha utilizado previamente la Consola de nube para crear una cuenta de facturación y / o un proyecto. Si es un usuario actual, consulte los Procedimientos iniciales . (Si intenta utilizar el Inicio rápido, será redirigido a la Consola de nube).

Inicio rápido

Nota: Solo para usuarios nuevos.

Para comenzar, haga clic en el botón Comenzar a continuación. Se lo guiará a (1) elegir uno o más productos, (2) seleccionar o crear un proyecto, (3) configurar una cuenta de facturación y habilitar las API asociadas con los productos que eligió.

Empezar

Al final del proceso, se creará una clave API para usted. Para agregar y restringir su clave API, consulte Obtener, Agregar y restringir una clave API .

Procedimientos iniciales

Nota: para usuarios nuevos o usuarios actuales.

Las instrucciones en esta sección describen cómo usar la Consola de la nube para realizar los procedimientos de inicio.

Nota: A medida que continúe su trabajo de desarrollo, puede regresar a la Consola de la nube para realizar cualquiera de estos procedimientos, según sea necesario (por ejemplo, puede crear una cuenta de facturación adicional u obtener una segunda clave API).
